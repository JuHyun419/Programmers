# 문제 설명<br>
#### 길이가 n, "수박수박수박수..."와 같은 패턴을 유지하는 문자열을 리턴하는 함수 작성
#### n이 4이면 "수박수박", n이 3이면 "수박수"를 리턴<br><br><br>
# 제한 사항<br>
####
- n은 길이 10,000 이하인 자연수<br><br><br>
# 입출력 예  
| n | return | 
---|:---:
| 3 | "수박수" | 
| 4 | "수박수박" |

```java
class Solution {
  public String solution(int n) {
      String answer = "";
      for(int i=1; i<=n; i++)
      {
          if(i%2 == 0)
          {
              answer += "박";
          }
          else
          {
              answer += "수";
          }
      }
      return answer;
  }
}
```
